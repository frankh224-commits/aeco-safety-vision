# SAM Exploration Notes

Segment Anything Model (SAM) was explored as part of dataset experimentation.

Observations:

* SAM can generate segmentation masks quickly.
* Useful for rapid annotation assistance.

Limitations:

* Masks often require manual correction.
* Not optimized for detecting small safety equipment such as helmets in crowded scenes.

Conclusion:

SAM was considered helpful for annotation exploration but YOLO object detection was selected as the main modeling approach for this prototype.
